## How to use

First clone the repo into your disk.

```bash
$ git clone https://github.com/rootlulu/shell_stock_display.git
```

then

```bash

$ chmod +x run.sh

$ ./run.sh
```

## How to add your stocks

Edit stock_codes, just put stock codes here, like this:
